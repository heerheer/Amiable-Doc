# SetAppInfo()

Amiable的设置应用信息的逻辑。

需要在这里设置您的应用信息

当然:我们推荐你将应用AppInfo数据写入另外的类库项目中，在AmiableService.cs中直接调用函数。

这样做可以方便使用一个解决方案多个插件项目 。 同时避免忘记在Core中使用其他程序集的类导致反射加载事件失败。

例如

App.AppInfo = ReplyBot.GetAppInfo();
//App.AppInfo = HorseRaceBot.GetAppInfo();
