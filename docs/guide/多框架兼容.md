# 多框架兼容

## Adapter

适配器是对API的兼容

### 安装

使用**Nuget**或者直接引用**源码**中的项目。

### 扩展方法

适配器会提供一个针对`AppService`的扩展方法

你可以在[适配器]()查看目前支持的适配器

## ServiceBuilder

对`AppService`使用不同的扩展方法

```
public static void ServiceBuilder(AppService service)
{
    //使用MQ配置
    service.UseMQConfig();
}
```

## 编译符号

Amiable.Core项目的Build生成配置中

你可以在[支持的平台]()查看目前支持的平台，以更改编译符号来编译针对不同平台的插件
